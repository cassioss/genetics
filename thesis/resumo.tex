A extração de características musicais de sinais sonoros é uma tarefa com aplicação em diversas áreas como compressão de sinais, ensino, edição musical, etc. Todavia, extrair informações de um sinal sonoro gravado com instrumentos reais é uma tarefa complexa. Apesar de diversos métodos já terem sido desenvolvidos para extração de características como tom, intensidade e duração de notas, existem poucos trabalhos que unificam essas etapas com alta confiabilidade, o que se reflete na escassez de produtos comerciais capazes de interpretar musicalmente sinais sonoros de forma fiel. Este trabalho objetiva a implementação de um programa capaz de realizar em tempo real a transcrição musical de sinais sonoros, mesmo que ruidosos, para notação musical moderna. Para tornar o problema tratável, o estudo foi desenvolvido visando a análise de sinais provenientes de instrumentos de sopro temperados e monofônicos, embora também sejam apresentados resultados obtidos ao analisar outros tipos de instrumento. O programa desenvolvido pode ser dividido em 3 módulos principais, responsáveis pela análise de tom, intensidade e tempo. A extração de tom é feita utilizando uma adaptação do algoritmo de produto espectral harmônico (HPS), a de intensidade pelo cálculo da raiz quadrática média e a de tempo por algoritmos de aprendizado. Por fim um método heurístico utiliza os resultados dos 3 módulos para gerar um arquivo MIDI para o sinal, que pode ser facilmente convertido para notação musical moderna. O programa desenvolvido foi avaliado utilizando músicas do repertório clássico ocidental.
